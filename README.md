# Fake-Review-Detection
An AI-powered Fake Product Review Detection System that uses Natural Language Processing (NLP), TF-IDF feature extraction, Logistic Regression, and Explainable AI (XAI) to identify deceptive reviews. Built with Streamlit, it provides an interactive, transparent, and user-friendly web application for combating review fraud in e-commerce.
